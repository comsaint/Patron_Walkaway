# Patron_Walkaway
Patron Walkaway Detection project. 

Our mass gaming floor has deployed Smart Table technology through which we are able to capture the betting behavior of every patron (rated or not) in real-time. The goal is to detect in real-time whether a gaming patron will stop gaming and leave in upcoming 15 minutes, so that our hosts can approach and retain him/her.
